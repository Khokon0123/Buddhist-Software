# Bengali Fonts

Please download and place the following Bengali fonts in this directory:

1. **NotoSansBengali-Regular.ttf** - Download from: https://fonts.google.com/noto/specimen/Noto+Sans+Bengali
2. **NotoSansBengali-Bold.ttf** - Download from: https://fonts.google.com/noto/specimen/Noto+Sans+Bengali
3. **NotoSerifBengali-Regular.ttf** - Download from: https://fonts.google.com/noto/specimen/Noto+Serif+Bengali
4. **NotoSerifBengali-Bold.ttf** - Download from: https://fonts.google.com/noto/specimen/Noto+Serif+Bengali

Alternatively, you can use any Bengali font that supports Unicode Bengali characters.
